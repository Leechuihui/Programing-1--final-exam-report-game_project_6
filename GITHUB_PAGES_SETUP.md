# GitHub Pages 设置指南

## 🎯 启用GitHub Pages的详细步骤

### 步骤1：访问仓库设置页面
1. 打开浏览器，访问：https://github.com/Leechuihui/Programing-1--final-exam-report-game_project_6
2. 点击仓库页面顶部的 **"Settings"** 标签页

### 步骤2：找到Pages设置
1. 在左侧菜单中，向下滚动找到 **"Pages"** 选项
2. 点击 **"Pages"** 进入页面设置

### 步骤3：配置Source
1. 在 **"Source"** 部分，您会看到三个选项：
   - Deploy from a branch
   - GitHub Actions
   - Deploy from a branch (Legacy)

2. **选择 "GitHub Actions"** （这是新的推荐方式）

### 步骤4：等待工作流运行
1. 选择GitHub Actions后，系统会自动检测到我们配置的工作流
2. 您会看到工作流开始运行（通常需要2-3分钟）

### 步骤5：验证部署
1. 部署完成后，您会看到绿色的勾号 ✅
2. 您的游戏将在以下地址上线：
   **https://Leechuihui.github.io/Programing-1--final-exam-report-game_project_6**

## 🔍 如果找不到Pages选项

### 可能的原因：
1. **仓库是私有的** - GitHub Pages需要公开仓库
2. **权限不足** - 确保您是仓库的所有者
3. **页面加载问题** - 尝试刷新页面

### 解决方案：
1. 确保仓库是公开的（Public）
2. 检查您是否有管理员权限
3. 清除浏览器缓存并重新访问

## 📱 移动端设置
如果您在手机上操作：
1. 点击仓库页面右上角的 **"..."** 菜单
2. 选择 **"Settings"**
3. 向下滚动找到 **"Pages"**
