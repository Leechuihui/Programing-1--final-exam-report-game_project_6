## 📋 How to Take Screenshots

1. **Start the game**: Open the live demo or run locally
2. **Navigate scenes**: Use game controls to reach desired scenes
3. **Capture moments**: Use screenshot tools to capture key moments
4. **Save with descriptive names**: Use the naming convention above
5. **Optimize size**: Keep files under 2MB each
