# Black Cat Granary Saga: Dream Prologue v1.0.0

## 🎉 First Official Release

This is the first official release of **Black Cat Granary Saga: Dream Prologue**, a heartfelt 2D platformer game developed as a final programming assignment for the University of London Computer Science undergraduate program.

## 💝 What Makes This Special

This game is more than just a programming project - it's a digital memorial to a beloved childhood mountain cat companion who brought warmth, joy, and unconditional love during six precious years. Every element of the game reflects the deep bond between a child and their most treasured feline friend.

## ✨ Features Included

### 🎮 Core Gameplay
- **Smooth 2D Platformer Mechanics**: Precise jump controls and responsive movement
- **Dynamic Weather System**: Day/night cycles, rain, snow, and lightning effects
- **Interactive Environment**: Cloud platforms, diverse terrain, and collectible items
- **Enemy AI System**: Intelligent mouse enemies with different behaviors
- **Power-up System**: Speed boost, jump enhancement, shield, and magnet abilities
- **Scoring System**: Points, combos, and achievement tracking

### 🎨 Visual & Audio
- **Particle Effects**: Rich visual effects and animations
- **Starry Sky Background**: Dynamic starfield and meteor effects
- **Audio Integration**: Background music and sound effects
- **Responsive Design**: Works on different screen sizes

### 🛠️ Technical Features
- **Modular Architecture**: Clean, organized code structure
- **GitHub Pages Ready**: Automatic deployment configuration
- **Cross-browser Compatible**: Works on modern web browsers
- **Mobile Friendly**: Touch controls and responsive layout

## 🎯 Game Story

Late autumn, 1970, 5:24 PM. Outside the village granary, the black cat Shadowpaw gazes into the distance. The Mouse King and his army have been stealing food, leaving villagers hungry. The village chief hands Shadowpaw his last cornbread, hoping he can protect the granary and restore hope.

As night falls, the wind carries whispers of the mouse horde. Shadowpaw tightens his cloak and embarks on his journey of defense...

## 🚀 How to Play

### Online Version
- **Live Demo**: https://Leechuihui.github.io/Programing-1--final-exam-report-game_project_6
- **GitHub Repository**: https://github.com/Leechuihui/Programing-1--final-exam-report-game_project_6

### Local Installation
1. Clone the repository
2. Open `index.html` in a web browser
3. Or use a local server: `python -m http.server 8000`

### Controls
- **WASD** or **Arrow Keys**: Move
- **Spacebar**: Jump
- **Mouse Click**: Jump
- **P**: Pause/Resume
- **R**: Restart
- **M**: Toggle audio
- **N**: Toggle day/night
- **G**: Toggle snow

## 📋 System Requirements

- **Browser**: Modern web browser (Chrome, Firefox, Safari, Edge)
- **JavaScript**: ES6+ support required
- **Internet**: Required for CDN resources (p5.js, p5.sound)

## 🎓 Academic Context

This project was developed as the final programming assignment for **Programming 1** in the University of London Computer Science undergraduate program. It demonstrates:

- Object-oriented programming principles
- Game development concepts
- Web technologies (HTML5, CSS3, JavaScript)
- Version control with Git
- Project documentation and presentation
